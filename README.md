가위바위보 게임  ( 2명 )
 - 기능
   = 컴퓨터와 게임
   = 삼세판 설정
   = 통계 ( 몇번 이겼는지) 
   = 승률보기 (ex: 주먹 30%, 바위 50%, 보 20% 로 이김)
   = 게임머니 기능 
      ~ 시작하자마자 게임머니 설정
      ~ 게임할때마다 자신이 보유한 게임을 판돈으로 건다. 이길경우 2배 또는 3배수로 받는다.
      ~ 둘중 게임머니가 0 이면 종료

   = 게임 이어서 하기
   = 현재 승패 및 게임머니 파일로 저장, 불러오기

 - 관련문법
    = random 으로 컴퓨터의 가위바위보 정하기
    = 기능은 함수로 구현
    = main 함수에서 while 루프를 이용한 게임이어서 하기
    = file 입출력
